Bill of materials for http://maven.apache.org/[Maven]-based projects. Use it as follows:

[source,xml]
----
<dependencyManagement>
  <dependencies>
    <dependency>
      <groupId>com.github.sebhoss.bom</groupId>
      <artifactId>${kind}-bom</artifactId>
      <version>${version.bom}</version>
      <type>pom</type>
      <scope>import</scope>
    </dependency>
  </dependencies>
</dependencyManagement>
----
